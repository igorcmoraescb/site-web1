Entrega 4

Faça isso depois de corrigir os erros anteriores!

Primeiro passo: Vocês devem começar a esboçar o menu do site de vocês. Para isso, vocês devem colocar todas as páginas do site de vocês linkadas em uma lista no início de cada página. Três páginas todos vocês vão ter no site de vocês, são elas: uma página inicial, uma de galeria e uma para contato (vocês não precisam colocar nenhum conteúdo nessa de contato além do menu). Vocês devem obrigatoriamente ter nesse menu uma estrutura de submenu, então além dessas páginas obrigatórias vocês vão ter que criar outras páginas para cumprir esse requisito. Tudo deve ter um sentido lógico para uso de submenu, então vou avaliar também o contexto do site de vocês. Lembrem-se de configurar corretamente a ordem de navegação (next e prev).

Segundo passo: A(s) lista(s) que vocês já haviam criado devem ser corrigidas colocando o type, e o(s) link(s) externos devem ser corrigidos adicionando os parâmetros target e rel com os valores corretos para esse tipo de link.

Terceiro passo: Adicionem um arquivo para download e um audio. Lembrando sempre que todo conteúdo novo deve estar inserido em um contexto.

Quarto passo: Todas as imagens do site de vocês devem agora ter tamanhos variáveis para diferentes tipos de tela. Siga o mesmo padrão que eu estou utilizando no meu site.

O que é cobrado nesse enunciado é só o mínimo para que vocês pontuem, mas sintam-se à vontade para adicionar mais conteúdo (Desde que esteja certo).
