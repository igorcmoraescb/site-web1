Isso aqui é um subtítulo que eu posso usar por exemplo para detalhar brevemente o assunto do título.

Explicação das tags

A hierarquia de títulos vai do h1 ao h6. Não é permitido usar um título de hierarquia mais baixa sem que tenha ocorrências na mesma página de outros títulos de hierarquia mais alta.

Geralmente essas tags são utilizadas para divisão de seções de um artigo, por exemplo as páginas da Wikipedia. Outro uso recorrente das tags h, especificamente as h1 e h2, é em sites de notícias. É muito comum esse tipo de site ter o título de uma notícia e embaixo uma frase que aborda o tema mais importante dela.

Além das tags h, outra tag que vimos é a tag p, que cria parágrafos. O resto das tags são mais como tags de configuração e estrutura de código: html, head, title e body.

Enunciado da atividade 1

Para essa atividade, vocês devem começar a criação da página inicial do site. Toda página inicial deve ter o mesmo nome de arquivo, index.html

Utilize as tags de estrutura e configuração, e depois disso crie um conteúdo textual com base no tema desse site. Vocês devem utilizar de forma correta e coerente seguindo os seguintes requisitos: no mínimo usar as tags h1 a h3 e também 3 parágrafos. O conteúdo pode usar mais tags h e ter mais parágrafos, desde que mantenha a coerência semântica do uso dessas tags.
